## 📊 **数据源总结**

#### **Polymarket Gamma API**
**API**: https://gamma-api.polymarket.com/events

**重要发现**：
- ⚠️ 没有 "breaking" 参数
- ✅ 按 volume24hr 排序可获取热门事件
- ✅ 数据在 `markets` 数组中
- ✅ `outcomePrices` 格式：["Yes价格", "No价格"]
- ✅ 价格转换为概率：`float(prices[0]) * 100`
- ⚠️ API 数据有延迟，可能几分钟到几十分钟不等

**实际用法**：
```bash
curl "https://gamma-api.polymarket.com/events?active=true&limit=10"
```

**事件 vs 市场**：
- **Event（事件）** - 例如："2026 NBA Champion"
- **Market（市场）** - 例如："Will 2026 NBA Champion win?"（市场问题）
